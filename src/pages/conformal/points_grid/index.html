Complex Function Transformations

z = x + i·y
z² = x² - y² + i·2xy
1/z = x/(x² + y²) - i·y/(x² + y²)
z³ = x³ - 3xy² + i·(3x²y - y³)
sin(z) = sin(x)cosh(y) + i·cos(x)sinh(y)
cos(z) = cos(x)cosh(y) - i·sin(x)sinh(y)
tan(z) = sin(2x)/(cos(2x)+cosh(2y)) + i·sinh(2y)/(cos(2x)+cosh(2y))
log(z) = ln|z| + i·Arg(z) = ln(√(x²+y²)) + i·atan2(y, x)
ln(z) = ln|z| + i·Arg(z) = ln(√(x²+y²)) + i·atan2(y, x)
e^z = e^x(cos(y) + i·sin(y))
